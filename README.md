# Frostmoln Homebrew Tap

Homebrew tap for the [Frostmoln](https://frostmoln.cloud) CLI, `fm`.

## Install

```sh
brew install frostmoln/tap/fm
```

or

```sh
brew tap frostmoln/tap
brew install fm
```

## Notes

- `Formula/` is generated automatically by goreleaser on each `fm` release. Do not edit by hand.
- Binaries are hosted in the EU on `get.frostmoln.cloud`; this repo holds only the formula.
- Verify releases with minisign (public key + instructions in the Frostmoln docs).
